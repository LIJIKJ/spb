# spb - Simple PasteBin

## What?

This is a simple pastebin made with `node.js` &amp; a `mongodb` backend.

## Why ?

I needed my own pastebin for "research" ;)

## How ?

- clone the repo
- take a look at [config.js](./config.js)
- `$ npm install`
- have mongodb running
- `$ npm start`
- pop open `http://localhost:3030` on the browser
